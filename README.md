# AtCoder Solved Problems

AtCoder（ABC）で過去に解いたAC（正解）コードを管理しているリポジトリです。

## 💡 特徴・仕様
- **解答コードのみ自動同期**: 手元の環境を汚さないよう、未解答（テンプレート状態）のファイルやテストケースはGit側で自動的に除外。実際に `>>` を使って解答を書き込んだ `main.cpp` とその解説用 `README.md` のみが公開されるように制御しています。
- **問題リンクの自動生成**: 各問題フォルダ内には、`atcoder-cli` の拡張機能により、問題へのリンクが含まれた `README.md` が自動生成されます。

## 🛠️ 動作環境
- WSL2 (Ubuntu)
- `atcoder-cli` (acc)
- `online-judge-tools` (oj)

## ✍️ 今後の精進メモ
- 後で見返したい神問題や、詰まったコーナーケースなどは、各フォルダの `README.md` に解法や考察を追記していく。
